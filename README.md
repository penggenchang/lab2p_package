# lab2p
A user-friendly, Python-based, one-click run for non-programmers to process and analyze 2P data. 

Developed for Latifi Lab. PI: Dr. Latifi. [Contact](shahrzad.latifikhereshky@hsc.wvu.edu). 

---

## Features
- Batch Suite2p processing
- ROI validation
- Traces heatmaps
- ROI summary table
- dF/F and normalized deconvolved spike
- Pearson correlation and functional connectivity
